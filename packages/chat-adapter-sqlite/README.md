Beta SQLite state adapter for [Chat SDK](https://chat-sdk.dev/docs) built with [better-sqlite3](https://github.com/WiseLibs/better-sqlite3). Use this when SQLite is your primary datastore and you want state persistence without a separate Redis dependency.

## 安装

```bash
npm install chat chat-adapter-sqlite
```
